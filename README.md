\# Nourishland XR Platform



A reusable spatial learning platform for:



\- Botanical Gardens

\- Food Forests

\- Public Parks

\- Universities

\- Museums



\## Principles



\- One engine

\- Many sites

\- One content structure

\- Multiple devices



\## First Demonstration



Hillyards Food Forest



The first demonstration site used to develop and validate the platform.



\## Project Structure



\- app

\- engine

\- content

\- sites

\- templates

\- examples

\- media

\- docs

\- schemas
